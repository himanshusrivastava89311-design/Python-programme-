#Python-programme-
🚀Description
This is a simple Python program that calculates the area of a circle based on the radius entered by the user. It uses the basic mathematical formula:
Area = π × r²
The program takes input from the user and prints the calculated area.
⚙️ How It Works
User enters the radius of the circle
Program calculates the area using value of π = 3.14
Displays the result
Code
r = float(input("enter the radius:"))
area = 3.14 * r * r

print(area)
How to Run
Install Python (if not already installed)
Save the file as area_of_circle.py
Open terminal / command prompt
Run the program:

User Input (input())
Type Casting (float())
